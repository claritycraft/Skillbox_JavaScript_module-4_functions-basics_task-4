### En:
#### Task 4
Task Description
Create a function that adds +1 to a global variable count and outputs the result to the console.

Example
The program has a variable defined as:
count = 3
Call: countPlusOne()
Output: 4
Call: countPlusOne()
Output: 5

#### Задача 4

Пример:

В программе записано значение переменной count = 3

Вызов: countPlusOne()

Вывод: 4
